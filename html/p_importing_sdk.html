How to import the SDK

Multiple programming languages are supported and therefore there are some differences in importing and using the SDK in these languages. Below you will find the steps you need to take in order to use the SDK in a project of the chosen programming language. Note that having a look at the examples (see Code Examples) and the README documents there might be an easier way to gain a better understanding of how to import the SDK. Next to this, be sure to have a look at the requirements and prerequisites before importing the SDK.

Importing in a C/C++ Project

  1. Add the include directory with the headers of the Xsens DOT PC SDK to your project.
  2. Link the libraries to your project (xsensdot_pc_sdk64 and xstypes64)
  3. Include the xsensdot_pc_sdk and xstypes headers
  4. Build and run your project, you should now be able to use the functionalities of the Xsens DOT PC SDK.
    • Add _CRT_SECURE_NO_WARNINGS to your code or compiler flags to avoid deprecation warnings from the SDK.

Importing in a Python Project

Importing the SDK in Python just requires a few simple steps. Assuming you have Python 3.7+ installed and pip available you need to follow the following steps.

  1. Install the requirements using pip, the requirements file can be found in the examples folder (pip install -r requirements.txt)
  2. Install the wheel file for your version of Python (pip install xsensdot_pc_sdk-*version*.whl)
  3. Now in your Python script add a line to import the SDK (import xsensdot_pc_sdk)
  4. Xsens DOT PC SDK should be ready for use in your project now.

Importing in a Java Project

  1. Import all the java wrapper files from the xsensdot_pc_sdk folder in your Java project. (import com.xsens.xsensdot_pc_sdk.*;)
  2. Make sure to load the required Library
    • Add the library folder to your PATH environment variable and use "System.loadLibrary("xsensdot_pc_sdk_java64");"
    • Java needs to be able to find it on the classpath or you will get errors that dependent libraries can not be found
    • The library files are located in the 'java' folder.
      • Visual Studio Code; Opening the project from the java folder makes sure the files are in the current directory.
      • Linux shell: Setting LD_LIBRARY_PATH to the java folder is an option to make sure the libraries can be found.
    1. Build and Run your code, you should now be able to use the functionalities of the Xsens DOT PC SDK.

Importing in a C# Project

  1. Import the C# wrapper classes from the wrap_csharp64 Folder
  2. Make sure your project can find the required library files (DLL or SO files)
  3. Add the "using XsensDotPcSdk;" to your code
  4. Build and run your project, you should now be able to use the functionalities of the Xsens DOT PC SDK.
